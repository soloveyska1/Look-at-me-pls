## 🧪 A/B TESTING PROTOCOL

### TEST 1: ВОКАЛЬНАЯ РАЗБОРЧИВОСТЬ

```
┌──────────────────────────────────────────────────────────────┐
│  МЕТОД: Solo вокал, затем включи бит                        │
├──────────────────────────────────────────────────────────────┤
│  ПРОВЕРКИ:                                                   │
│  □ Слова понятны на всех частотах?                          │
│  □ Вокал не "тонет" в миксе?                                │
│  □ Консонанты (S, T, K) четкие?                             │
│  □ Вокал "сидит" на бите или "плавает"?                     │
│                                                              │
│  ИНСТРУМЕНТЫ:                                                │
│  ├─ SPAN (Voxengo): Смотри overlap вокала и бита            │
│  ├─ Correlation Meter: Проверь phase issues                 │
│  └─ Level Meter: Вокал должен быть -12 to -8 dBFS           │
│                                                              │
│  ЦЕЛЕВОЙ РЕЗУЛЬТАТ:                                          │
│  ✓ Вокал на 3-5 dB громче в миксе                           │
│  ✓ Нет маскировки на 200-800 Hz                             │
│  ✓ Вокал четкий в моно и стерео                             │
└──────────────────────────────────────────────────────────────┘
```

│  □ Stereo width сравнима?                                   │
